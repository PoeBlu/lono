## Usage

    lono code convert SOURCE

## Description

Converts snippet of YAML to lono dsl format.

## Example

    lono code convert path/to/file


## Options

```
[--casing=CASING]  # Controls casing of logical ids. IE: as-is, camelcase or underscore
                   # Default: as-is
```
