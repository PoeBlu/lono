## Usage

    lono param generate

## Description

Generate all parameter files to `output/params`.

## Example

To generate CloudFormation json parameter files in the `output` folder:

    lono param generate


## Options

```
[--verbose], [--no-verbose]  
[--noop], [--no-noop]        
[--mute], [--no-mute]        
```
