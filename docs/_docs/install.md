### RubyGems

You can also install lono via RubyGems.

```sh
gem install lono
```

Or you can add lono to your Gemfile in your project if you are working with a ruby project.  It is not required for your project to be a ruby project to use lono.

{% highlight ruby %}
gem "lono"
{% endhighlight %}
