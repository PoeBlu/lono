## Example

  lono completion

Prints words for TAB auto-completion.

## Examples

  lono completion
  lono completion cfn
  lono completion cfn create

To enable, TAB auto-completion add the following to your profile:

  eval $(lono completion script)

Auto-completion example usage:

  lono [TAB]
  lono cfn [TAB]
  lono cfn create [TAB]
  lono cfn create --[TAB]
